# Profile Card Generator

A modern Flutter application for creating beautiful, customizable profile cards in minutes. Personalize your details, preview the design in real time, and export or share polished profile cards with ease.

## Features

* Clean and responsive UI
* Customizable profile information
* Real-time card preview
* Modern card layouts
* Easy export and sharing *(coming soon)*

## Tech Stack

* Flutter
* Dart
* Material Design

## Getting Started

```bash
git clone https://github.com/indeedSunil/profile-card-generator.git
cd profile-card-generator
flutter pub get
flutter run
```
